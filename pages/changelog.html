Changelog

v0.3 - Options Menu

  • Added a "More Options" menu in the Taps screen.
  • This is still being tested, keeping powers the same until we're sure it didn't break anything
  • Added a "Max Sauce" option.
  • This will allow users to choose how much sauce to simulate, which will allow for higher sauce amounts to be simulated faster.
  • Go crazy on 10t tap sims :D
  • Added a "Help" menu to explain what the options do
  • All options have default values which are shown in the help menu and can be set with the appropriate button
  • Added titles to the versions

v0.2 - Graph Update

  • Expected values now appear instantly instead of updating during the simulation
  • Graph now shows only relevant data
  • Now shows eggs which were expected but not gotten
  • No longer shows eggs which were not gotten nor expected
  • Graph is now organized by rarity, no more seeing 10t before 5t!
  • Added this handy-dandy changelog. Isn't it neat?

v0.1 - First Version

  • Added taps simulation
  • Added powers simulation
  • Added graph to simulations to better visualize simulation data
  • Expected values added to taps simulation
  • Added navigation bar as well as footer
  • User can now use abbreviations for large numbers (1m instead of 1000000)
